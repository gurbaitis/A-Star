Description
-----------
The AStar program finds the fewest amount of one character changes between
two words using an inputted dictionary via the A* algorithm. The start word,
end word, and each word with 1 character change in between is printed to the console.

Usage
-----
To run this program, use the following commands:

    java -jar Lab1AStar_Gabriel_Urbaitis.jar [textfile.txt] [String] [String]
